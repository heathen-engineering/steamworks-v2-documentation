Detailed Description

Abstract structure for game data models.

Used to create data models suitable for save and load operations via RemoteStorageSystem

See DataModel<T> for more informaiton on the usage of DataModel

This is an abstract class and must be inherited from to create a unique data model class suitable for your game. The use is similar to that of UnityEngine's UnityEvent<T> in that you can you create any data structure you like as a class or struct assuming of course that it is marked as [Serializable]. You can then declare a class and derive from DataModel<T> as demonstrated below. Note that DataModel is derived from Unity's ScriptableObject allowing you to create your data model object as an asset in your project

[Serializable]
public class MyCharacterData
{
public string characterName;
public int level;
public Serializable.SerializableVector3 position;
public Serializable.SerializableQuaternion rotation;
}
[CreateAssetMenu(menuName = "My Objects/Character Data Model")]
public class CharacterDataModel : DataModel<MyCharacterData> { }
