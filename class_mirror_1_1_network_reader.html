Detailed Description

Binary stream Reader.

Supports simple types, buffers, arrays, structs, and nested types

Use NetworkReaderPool.GetReader to reduce memory allocation

Note: This class is intended to be extremely pedantic, and throw exceptions whenever stuff is going slightly wrong. The exceptions will be handled in NetworkServer/NetworkClient.
