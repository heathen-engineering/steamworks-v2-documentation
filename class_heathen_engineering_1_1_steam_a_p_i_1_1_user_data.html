Detailed Description

Represents the sum of a users persona information as defined by Steamworks.

The HeathenEngineering.Steam.Foundation.SteamUserData object provides access and real time update of a given user's display name, avatar, game state informaiton and persona state information.

This object should not be created manually but rather looked up or generated by calling HeathenEngineering.Steam.Foundation.SteamSettings.GetUserData(CSteamID)

You should attempt to sync UserData objects over the network. It is more efficent to sent the ulong value of the user's ID instead for example

userData.id.m_SteamID;

This value can then be casts to a CSteamID or used on another client ot fetch the UserData object that matches the id

  • To fetch the local user's HeathenEngineering.Steam.UserData object you can the userData member of HeathenEngineering.Steam.Foundation.SteamSettings.client, however for demonstration purposes this is how you would look it up by CSteamID
    var localUser = settings.GetUserData(SteamUser.SteamId.m_SteamID);
    Debug.Log("The local user's Steamworks display name is " + localUser.DisplayName);

◆ GameInfo

FriendGameInfo_t HeathenEngineering.SteamAPI.UserData.GameInfo
get

What is the users current game info if any

https://partner.steamgames.com/doc/api/ISteamFriends#FriendGameInfo_t

This calls SteamFriends.GetFriendGamePlayed(CSteamID, out FriendGameInfo_t) so you should cashe the result in a local variable and use that to perform any additional tests ... e.g.

//Do not do this
var gameId = userData.GameInfo.m_gameID;
var lobbyId = userData.GameInfo.m_steamIDLobby;
//Instead do this
var gameInfo = userData.GameInfo;
var gameId = gameInfo.m_gameID;
var lobbyId = gameInfo.m_steamIDLobby;

◆ State

EPersonaState HeathenEngineering.SteamAPI.UserData.State
get

The current persona state of this user

https://partner.steamgames.com/doc/api/ISteamFriends#EPersonaState

  • k_EPersonaStateOffline 0 Friend is not currently logged on.
  • k_EPersonaStateOnline 1 Friend is logged on.
  • k_EPersonaStateBusy 2 User is on, but busy.
  • k_EPersonaStateAway 3 Auto-away feature.
  • k_EPersonaStateSnooze 4 Auto-away for a long time.
  • k_EPersonaStateLookingToTrade 5 Online, trading..
  • k_EPersonaStateLookingToPlay 6 Online, wanting to play.
  • k_EPersonaStateMax 7 The total number of states. Only used for looping and validation.
