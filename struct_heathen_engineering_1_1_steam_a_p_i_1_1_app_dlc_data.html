Detailed Description

Represents the metadat of a DLC entry for a given app This is returned by HeathenEngineering.Steam.Utilities.GetDLCData

  • Return the list of all DLC for the current application
    var results = SteamUtilities.GetDLCData();
    foreach(var result in results)
    {
    Debug.Log("Located DLC " + result.name " with an AppId of " + result.appId.m_AppId.ToString() + ", this DLC is " + (result.available ? "available!" : "not available!"));
    }
