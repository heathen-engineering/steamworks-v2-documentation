Detailed Description

Helps query for servers on the Steam Game Server system

Simplifies the process of querying for Steam Game Servers and stores the results of queries in InternetServers, FavoritesServers, FriendsServers, LANServers, SpectatorServers, and HistoryServers lists.

The data provdied by this object can be used to drive UI elements or facilitate matchmaking/server discovery. Specifics of records returned can be controlled via the filter member.
