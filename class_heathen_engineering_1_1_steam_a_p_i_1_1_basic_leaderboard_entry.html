Detailed Description

UI componenet to help display leaderboard entries

This can be used as is but is intended to serve as an example of how to create your own UI script for displaying leaderboard entry data. To create your own simply derive from HeathenSteamLeaderboardEntry as demonstrated in this class and apply your desired logic.
