Detailed Description

Gathers and stores information about a Steamworks Clan aka Steamworks Group

Steamworks Clans also know as Steamworks Groups are a social feature of the Steamworks API. You can fetch a list SteamClan objects for all the clans/groups a user belongs to via the GameClient.ListClans method accessable from the client member.

Returns a list of chat members in the clan chat.

Returns

This has some limitaitons as outlined in the Valve documetnation . In particular large groups will not be able to be iterated over so will return 0 or incomplete lists of users. It is strongly recomended that you use the OpenChat feature to open the Steamworks Overlay to the chat as opposed to trying to manage it in game.
